### Understanding Naming of CSV Files
File Starts With
- "GP": General Primary
- "GE": General Election
- "CE": Consolidated Election
- "SP": Special Primary
- "SE": Special General Election

File title continues with numbers in the format of YYYYMMDD.
- This represents the date when the election was.
